## Features

- Manage your game library
- Cloud save sync
- Achievements and profile
- Rich catalogue with smart suggestions
- Lightweight and easy to use

## Development

### Requirements

- Node.js + Yarn
- Python 3.x

### Install

```bash
yarn install
```

### Dev

```bash
yarn dev
```

### Build Windows

```bash
yarn build:win
```
